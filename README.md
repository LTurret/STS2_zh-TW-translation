# STS2_zh-TW-translation

Slay The Spire 2 繁體中文在地化翻譯

> [!IMPORTANT]
> 遊戲版本：`v.0.99.1`

## 使用方法

### Windows

1. 下載本專案最新的翻譯文件
2. 將本專案 `./localization_override/zhs` 資料夾放置於 `%APPDATA%\SlayTheSpire2\localization_override` 路徑下

## 詞語文本

中文段落中括號以根據[《重訂標點符號手冊》修訂版](https://language.moe.gov.tw/001/Upload/FILES/SITE_CONTENT/M0001/HAU/haushou.htm)將括號 `“”` 替換成 `「」`，英文段落則保留 `“”`。

並且針對以下詞語／錯別字／歧異，進行臺灣正體習慣用詞地化：

### 特殊情況

根據語境調整文本的詞彙：

- `SILENT2_EPOCH.description`

   > 另幾個姐妹們靜默地審視著第四位姐妹……
   > 她們已經有三個世紀多沒有見到她了，而後者帶回來的只是一塊醜陋的、搏動著的碎片。
   >
   > 靜默獵手回憶起在獲取這個戰利品的途中自己經歷的無數考驗，但她**始終**沒有開口——開口是一種弱小。要證明自己，**總是**只能靠更加輝煌的戰利品。
   >
   > 總是會有更大的獵物。
- `DOLL_ROOM.pages.MR_STRUGGLES.description`
   > 當你拾起那尊破損不堪的玩偶時，所有的喧囂都戛然而止。
   > 這一尊，**始終**緘默。
